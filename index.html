Content No Longer Available

25 August 2021

We regret to inform you that the content you were looking for is no longer available. As of 25 August 2021, the Lively website is no longer available online. After two successful years of operation, the Lively Discord server community was deleted on 1 September 2021. The Lively Discord community thanks the community and staff teams for making the community so memorable.

If you are in need urgent mental health assistance, please visit this list of suicide crisis lines. More mental health resources are available at Befrienders Worldwide. There are still many resources still available which will allow you to get the help you need.

In accordance with the GNU General Public License, the source code (excluding all brand assets) for this website remains publicly available on GitHub. This static error page should be equivocated to a 410 Gone HTTP error. The requested resource is no longer available on this server and there is no forwarding address. Please remove all references to this resource.
