Thank You

25 August 2021

Thank you for your support of the Lively mental health community since 2019. As of 25 August 2021, the Lively website is no longer available online. After two successful years of operation, Lively's Discord server was deleted on 1 September 2021. Lively thanks the community and staff teams for making the community so memorable.

If you are in need urgent of mental health assistance, please visit this list of suicide crisis lines. More mental health resources are available at Befrienders Worldwide. There are still many resources still available which will allow you to get the help you need.

In accordance with the GNU General Public License, the source code (excluding all brand assets) for this website remains publicly available on GitHub. This static error page should be equivocated to a 410 Gone HTTP error. The requested resource is no longer available on this server and there is no forwarding address. Please remove all references to this resource.

This status message is maintained by the former webmaster aidswidjaja and can be contacted at lively-webmaster@adrian.id.au. This email address should only be used for enquiries directly related to the operation of the Lively website and other IT assets, and the webmaster cannot provide information about the Lively community itself nor its closure.
